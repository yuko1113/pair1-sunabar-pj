# Usage

```bash
git clone https://github.com/yuko1113/pair1-sunabar-pj.git
```
```bash
docker-compose run -w /usr/src/app/next --rm app npm install
```
```bash
docker-compose up -d
```

After the application start, navigate to http://localhost:3000 in your browser.
